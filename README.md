# primary-math-lesson-plans-skill
专为小学数学教师设计，用于生成备课大纲、PPT课件和Word文档。当老师需要准备数学课教案、生成课件、编写教学设计、制作小学数学PPT、或导出Word版教案时触发。覆盖小学1-6年级所有数学知识点，如加减乘除、图形认识、分数、小数、应用题等。
小学数学备课技能
本技能为小学数学教师提供高效的备课支持，核心功能包括：生成标准化的备课大纲和生成配套的PPT课件。

适用场景
教师准备小学数学课需要生成教案大纲
需要制作小学数学课堂PPT课件
备课时间紧张需要快速生成标准教案
覆盖小学1-6年级所有数学知识点
核心功能
功能1：生成备课大纲
当教师输入具体数学知识点（如"小学三年级上册 两位数乘一位数"、"认识分数"、"平行四边形的面积"等），自动生成完整的备课大纲，包含六大核心模块：

教学目标：根据知识点设定知识与技能、过程与方法、情感态度与价值观三维目标
教学重难点：明确教学重点和难点
教学准备：课件、教具、学具等准备事项
教学过程：导入、新授、巩固、总结四个环节
课堂练习：针对本节课的随堂练习题
作业布置：分层作业设计
功能2：生成PPT课件
教师确认大纲后，发送"生成PPT"指令，自动生成适配小学数学课堂的PPT课件：

页面简洁清晰，适合小学生阅读
包含核心知识点、例题、练习
配色温馨活泼，符合儿童审美
动画适度，避免干扰注意力
功能3：导出Word文档
教师确认大纲后，可发送"导出Word"或"生成Word"指令，自动将备课大纲导出为结构化的Word文档：

完整表格化排版，八大模块清晰呈现
基本信息表：课题、教材版本、年级册次、单元、课时、教师、日期
教学目标三栏对比表（知识技能 / 过程方法 / 情感态度）
教学重难点双栏表
教学过程大表格（环节、时长、设计意图、具体内容）
课堂练习、作业布置多栏表
板书设计区域、教学反思三栏表
A4页面，可直接打印或提交
