# 小学数学备课技能

本技能为小学数学教师提供高效的备课支持，核心功能包括：生成标准化的备课大纲和生成配套的PPT课件。

## 适用场景

- 教师准备小学数学课需要生成教案大纲
- 需要制作小学数学课堂PPT课件
- 备课时间紧张需要快速生成标准教案
- 覆盖小学1-6年级所有数学知识点

## 核心功能

### 功能1：生成备课大纲

当教师输入具体数学知识点（如"小学三年级上册 两位数乘一位数"、"认识分数"、"平行四边形的面积"等），自动生成完整的备课大纲，包含六大核心模块：

1. **教学目标**：根据知识点设定知识与技能、过程与方法、情感态度与价值观三维目标
2. **教学重难点**：明确教学重点和难点
3. **教学准备**：课件、教具、学具等准备事项
4. **教学过程**：导入、新授、巩固、总结四个环节
5. **课堂练习**：针对本节课的随堂练习题
6. **作业布置**：分层作业设计

### 功能2：生成PPT课件

教师确认大纲后，发送"生成PPT"指令，自动生成适配小学数学课堂的PPT课件：

- 页面简洁清晰，适合小学生阅读
- 包含核心知识点、例题、练习
- 配色温馨活泼，符合儿童审美
- 动画适度，避免干扰注意力

### 功能3：导出Word文档

教师确认大纲后，可发送"导出Word"或"生成Word"指令，自动将备课大纲导出为结构化的Word文档：

- 完整表格化排版，八大模块清晰呈现
- 基本信息表：课题、教材版本、年级册次、单元、课时、教师、日期
- 教学目标三栏对比表（知识技能 / 过程方法 / 情感态度）
- 教学重难点双栏表
- 教学过程大表格（环节、时长、设计意图、具体内容）
- 课堂练习、作业布置多栏表
- 板书设计区域、教学反思三栏表
- A4页面，可直接打印或提交

## 使用流程

### 步骤1：接收并解析老师输入的知识点

1\. 识别用户输入中的"年级""数学知识点"，若信息不完整（如未说明年级），自动询问补充："请告知具体年级（1\-6年级），以便生成贴合该年级的备课内容哦\~"

2\. 通过web\_search工具，补充该知识点的教学要求（贴合小学数学新课标）、常见教学案例，确保备课内容符合教学规范。

### 步骤2：生成备课大纲

根据解析的知识点和教学要求，按固定结构生成备课大纲，每个模块贴合小学数学教学特点：

1. 教学目标：分"知识与技能""过程与方法""情感态度与价值观"三个维度，贴合小学生认知水平（如"能正确计算20以内退位减法，掌握'破十法''想加算减法'"）。

2. 教学重难点：重点明确知识点核心（如"两位数乘一位数的计算方法"），难点标注学生易出错的地方（如"进位的处理""理解比例的意义"）。

3. 教学准备：明确课堂所需教具（如小棒、计数器、图形卡片）、课件类型、练习单等。

4. 教学过程（40分钟课堂适配）：
        

    - 导入（5分钟）：设计趣味导入（如生活场景、小游戏、旧知回顾），吸引小学生注意力。

    - 新授（15分钟）：分步讲解知识点，结合简单例题，避免复杂表述，适配小学生理解节奏。

    - 巩固练习（12分钟）：设计基础题（全员必做）、提升题（选做），贴合课堂练习节奏。

    - 课堂总结（5分钟）：简洁梳理知识点，让学生明确核心内容。

5. 课堂练习：3\-5道贴合知识点的练习题，难度由浅入深。

6. 作业布置：分层作业（基础作业\+拓展作业），避免过重负担，贴合新课标要求。

3\. 输出备课大纲后，提示老师："备课大纲已生成！请选择下一步操作：输入『生成PPT』生成配套课件，或输入『导出Word』将大纲保存为Word文档哦~\~"

### 步骤3：接收老师优化意见，调整备课大纲

1\. 老师提出优化意见后（如"教学过程增加小组讨论""作业减少1道题"），自动修改备课大纲，修改后再次输出，供老师确认。

2\. 若老师无优化意见，直接确认大纲，进入输出环节。

## 步骤4：生成PPT课件或Word文档

### 4A：生成PPT课件

1. 调用Node.js + pptxgenjs，根据备课大纲生成PPT，每页内容对应大纲模块，排版简洁、字体清晰，适配小学生观看。

2. PPT页面结构（固定适配小学数学课堂）：


    - 第1页：标题页（年级、知识点、备课老师、日期）

    - 第2页：教学目标

    - 第3页：教学重难点

    - 第4\-8页：教学过程（按导入、新授、巩固、总结分步呈现，包含例题、小游戏、练习题干）

    - 第9页：课堂练习（完整题目）

    - 第10页：作业布置

    - 第11页：结束页（感谢观看）

3. PPT生成完成后，输出提示："PPT课件已生成，路径为：`~/skills/小学数学备课/ppt/XXX.pptx`，可直接打开使用~"

### 4B：导出Word文档

1. 调用Node.js + docx，根据当前备课大纲生成结构化Word文档，八大模块以表格形式呈现。

2. Word文档结构：

    - 基本信息表：课题、教材版本、年级册次、单元、课时、教师、日期

    - 教学目标三栏对比表（知识技能 / 过程方法 / 情感态度）

    - 教学重难点双栏表

    - 教学准备双栏表

    - 教学过程大表格（环节、时长、设计意图、具体内容）

    - 课堂练习双栏表

    - 作业布置三栏表

    - 板书设计区域

    - 教学反思三栏表

3. Word生成完成后，输出提示："Word文档已生成，路径为：`~/skills/小学数学备课/templates/lesson_plan_template.docx`，可直接下载打印使用~"

# 错误处理

- 异常1：老师输入的知识点不明确（如"数学 分数 备课"），未说明年级 → 提示："请补充具体年级（1\-6年级），以便生成贴合该年级的备课大纲和PPT哦\~"

- 异常2：生成PPT失败（如缺少依赖工具） → 提示："PPT生成失败，请检查Node.js环境，重新输入『生成PPT』即可~\~"

- 异常2b：生成Word文档失败 → 提示："Word文档生成失败，请重新输入『导出Word』尝试~\~"

- 异常3：老师输入的优化意见不清晰（如"修改教学过程"） → 提示："请具体说明修改内容（如'教学过程中导入环节改为小游戏'），我将精准调整备课大纲哦\~"

- 异常4：知识点超出小学数学范围 → 提示："抱歉，目前仅支持小学1\-6年级数学知识点备课，请输入正确的小学数学知识点哦\~"
